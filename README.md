# Contributing

To contribute, clone this repository.

## Clone the Repository

### Windows
Fork this repo, then run:
```bash
git clone https://github.com/Jessssswill/AI-flood.git
```

### Linux
```bash
git clone git@github.com:Jessssswill/AI-flood.git
```

## Requirements
1. ```npm install```
2. run ```npm start```
3. Enter ```localhost:3000``` in your browser
